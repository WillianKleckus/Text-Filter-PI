# Filtros Textuais

 ### Razão
 Trabalho de Processamento de Imagens da UFS. Programa consiste em:
 
  - Dilatar texto
  - Erodir texto
  - Com uma combinação das funções citadas, pode acabar com ruídos de sal e pimenta
  - Suavizar sal e pimenta com a máscara da média.
